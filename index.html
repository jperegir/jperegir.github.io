# SOBRE MÍ

Profesional con más de cinco años dedicados a la programación web y, desde hace más de tres, enfocado al desarrollo de aplicaciones empresariales en el sector logístico.

Con experiencia en integraciones de servicios web (API RESTful y SOAP), el desarrollo de aplicaciones web responsivas siguiendo el enfoque "mobile first" y la creación de contenido web y e-commerce (Woocommerce) mediante el sistema de gestión WordPress, entre otras cosas.

Trabajo diariamente con los lenguajes PHP, javascript y typescrypt, ejecutados sobre servidores Apache, en el primer caso, y sobre el entorno NODE.js y servidores Express.js, en el segundo y último caso.

Dispongo de amplia experiencia en el diseño de bases de datos relacionales, sobre todo definidas mediante el sistema de gestión Microsoft SQL Server.

Estoy muy interesado en el desarrollo de código limpio basado en los principios SOLID aplicados a la arquitectura hexagonal y en los Custom Elements aplicados al desarrollo de aplicaciones web reactivas. Además, estoy muy centrado en la adquisición de nuevos conocimientos CSS para poder desarrollar mejores interfaces gráficas en mis productos.

Recientemente, he adquirido conocimientos sobre el Framework Front End Angular 16, el cual estoy aplicando en mis últimos proyectos.
